# real-appeal

## Project setup

### Server setup
1. Add .env file to contain a reference to a MongoDB instance
2. Run the code below to begin the server:
```
nodemon app.js
```

### Client setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
